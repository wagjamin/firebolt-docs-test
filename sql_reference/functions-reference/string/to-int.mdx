Converts a string to an integer.

## Syntax


```sql
TO_INT(<expression>)
```

## Parameters


| Parameter | Description                         |Supported input types |
| :--------- | :----------------------------------- | :---------------------|
| `<expression>`  | The string to covert to an integer. | `TEXT` |

## Return Type
`INTEGER` 

## Example


The following example converts the input string to the integer `10`: 

```sql
SELECT
	TO_INT('10');
```

**Returns**: `10`
