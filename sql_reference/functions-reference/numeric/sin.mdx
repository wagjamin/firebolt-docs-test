Trigonometric function that calculates the sine of a provided value.

## Syntax


```sql
SIN(<value>)
```
## Parameters


| Parameter | Description     | Supported input types | 
| :--------- | :---------------------- | :----|
| `<value>`   | The value that determines the returned sine in radians. | `DOUBLE PRECISION` | 

## Return Type
`DOUBLE PRECISION`
