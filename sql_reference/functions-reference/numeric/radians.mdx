Converts degrees to radians.

## Syntax


```sql
RADIANS(<value>) 
```
## Parameters 


| Parameter | Description                                                                                                         | Supported input types | 
| :--------- | :------------------------------------------------------------------------------------------------------------------- | :------------| 
| `<value>`   | The value to be converted to radians | `DOUBLE PRECISION` |

## Return Type
`DOUBLE PRECISION`
