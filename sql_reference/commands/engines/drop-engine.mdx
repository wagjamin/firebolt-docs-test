Deletes an engine.

## Syntax

```sql
DROP ENGINE [IF EXISTS] <engine_name>
```
## Parameters 
   

| Parameter       | Description                           |
| :--------------- | :------------------------------------- |
| `<engine_name>` | The name of the engine to be deleted. |

## Example
The following example drops my_engine:

```sql
DROP ENGINE my_engine
```
