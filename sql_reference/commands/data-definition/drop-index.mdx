Deletes an index.

## Syntax

```sql
DROP AGGREGATING INDEX [IF EXISTS] <index_name>
```

## Parameters


| Parameter      | Description                          |
| :-------------- | :------------------------------------ |
| `<index_name>` | The name of the index to be deleted. |
