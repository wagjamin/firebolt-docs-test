Updates the specified role.

## ALTER ROLE OWNER TO

Change the owner of a role.

### Syntax

```sql
ALTER ROLE <role> OWNER TO <user>
```

### Parameters 


| Parameter | Description |
| :--- | :--- |
| `<role>` | Name of the role to change the owner of. |
| `<user>` | The new owner of the role. |
